#  ContestHub | Modern Contest Management Platform
 **ContestHub** is a robust and user-friendly platform designed to host, manage, and participate in various contests. Whether you're a creator looking to engage an audience or a participant aiming for the top of the leaderboard, ContestHub provides a seamless experience for all.

##  Key Features

###  Role-Based Dashboard
- **Admin:** Complete control over users and contests. Can block/unblock users and approve pending contests.
- **Creator:** Can create, edit, and manage their own contests, and see who submitted.
- **User:** Can explore contests, register, and track their participation status.

### Modern UI/UX
- **Dynamic Loading:** Global loading spinner for a smooth navigation experience.
- **Theme Toggle:** Switch between **Light** and **Dark** modes easily.
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop.

###  Secure Authentication
- Social Login (Google) and Email/Password-based authentication via **Firebase**.
- Secure private routes to protect sensitive data.

### Advanced Functionality
- **Search & Filter:** Find contests by category or name.
- **Leaderboard:** Track top performers across the platform.
- **Payment Integration:** Secure participation via **Stripe** (if implemented).
